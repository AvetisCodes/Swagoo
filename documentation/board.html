Board Usage

Board is similar to Jumbotron element in Bootstrap, but our name is better so we went with that.

    <div class="swagoo-board">
        <div class="swagoo-board-title">
            Public Annonucement
        </div>
        <div class="swagoo-board-subtitle">
            We just wanted to let you know that our framework is going to be updated periodically and you should be paying attention for more hanges to come.
        </div>
        <div class="swagoo-board-action">
            <button class="swagoo-btn">Learn More Here</button>
        </div>
    </div>


    <div class="row">
        <div class="col-1-1">
            <div class="swagoo-board">
                <div class="swagoo-board-title">
                    Public Annonucement
                </div>
                <div class="swagoo-board-subtitle rounded">
                    We just wanted to let you know that our framework is going to be updated periodically and you should be paying attention for more hanges to come.
                </div>
            </div>
        </div>
    </div>
