Display/Visibility Classes

Here is how the visibility classes work. You can show on 3 devices - mobile, tablet and desktop.

The way you specify what you would like to show is by adding the desired class for a specific device to the element.

For example: If you want elementA to be shown on mobile you will add the show class to it. Once you do elementA will ONLY be shown on mobile. If you decide you would like to show it on a tablet as well then you add another class for a tablet. Lastly, if you want it to be shown on all 3 types of devices - you could add the third class (for desktop in our case), however an easier way is not to put any visibility classes at all.

Lastly, you have choices on how you would like to display your elements - block, inline or inline-block.

Below are some examples: 

                    
    
        
        <div class="row">
            <div class="col-1-1">
                <h4 class="swagoo-show-block-on-desktop">Shown on desktop only as a block.</h4>
                <span class="swagoo-show-block-on-desktop">Content after it to demonstrate the block display.</span>
            </div>
        </div>

        <div class="row">
            <div class="col-1-1">
                <h4 class="swagoo-show-block-on-tablet">Shown on tablet only as a block.</h4>
                <span class="swagoo-show-block-on-tablet">Content after it to demonstrate the block display.</span>
            </div>
        </div>

        <div class="row">
            <div class="col-1-1">
                <h4 class="swagoo-show-block-on-mobile">Shown on mobile only as a block.</h4>
                <span class="swagoo-show-block-on-mobile">Content after it to demonstrate the block display.</span>
            </div>
        </div>

        <div class="row">
            <div class="col-1-1">
                <h4 class="swagoo-show-block-on-mobile swagoo-show-block-on-tablet">Shown on mobile and tablet only as a block.</h4>
                <span class="swagoo-show-block-on-mobile swagoo-show-block-on-tablet">Content after it to demonstrate the block display.</span>
            </div>
        </div>

        <div class="row">
            <div class="col-1-1">
                <h4 class="swagoo-show-block-on-mobile swagoo-show-block-on-desktop">Shown on mobile and desktop only as a block.</h4>
                <span class="swagoo-show-block-on-mobile swagoo-show-block-on-desktop">Content after it to demonstrate the block display.</span>
            </div>
        </div>

        <div class="row">
            <div class="col-1-1">
                <h4 class="swagoo-show-inline-on-mobile swagoo-show-inline-on-desktop">Shown on mobile and desktop only as inline.</h4>
                <span class="swagoo-show-inline-on-mobile swagoo-show-inline-on-desktop">Content after it to demonstrate the inline display.</span>
            </div>
        </div>

        <div class="row">
            <div class="col-1-1">
                <h4 class="swagoo-show-inline-block-on-mobile swagoo-show-inline-block-on-desktop">Shown on mobile and desktop only as inline-block.</h4>
                <span class="swagoo-show-inline-block-on-mobile swagoo-show-inline-block-on-desktop">Content after it to demonstrate the inline-block display.</span>
            </div>
        </div>
