Display/Visibility Classes

Here is how the visibility classes work. You can show on 3 devices - mobile, tablet and desktop.

The way you specify what you would like to show is by adding the desired class for a specific device to the element.

For example: If you want elementA to be shown on mobile you will add the show class to it. Once you do elementA will ONLY be shown on mobile. If you decide you would like to show it on a tablet as well then you add another class for a tablet. Lastly, if you want it to be shown on all 3 types of decides you could add the third class which would show it on the desktop, however an easier way is to not put any visibility classes at all.
