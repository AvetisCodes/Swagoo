Grid Combinations

Swagoo is a 6 column grid based framework and below you can find all of the combinations which you can use to build out your future website.

If you minimize the window you will see how the Grid responds to the changes.

            <div class="col-1-1">
                <p>class="col-1-1"</p>
            </div>
        </div>

        <div class="row">
            <div class="col-1-2">
                <p>class="col-1-2"</p>
            </div>
            <div class="col-1-2">

            </div>
        </div>
