Miscellaneous Elements

Even though this is the miscellaneous section there is still order to it - we have an HR element and 3 containers.

HR is just colorized and given a flat and more modern design which you can see on the front page.

The 3 container choices are a header, main container for body contant and lastly a footer.

We won't be showing their HTML equivalents on this page because you can already see how they look by looking at the current page's header, footer and main body container.

For the header - it just needs a title and a subtitle and a footer is a container for you final content - in my case it is a paragraph.

.swagoo-container is a bit different however, since you won't see any colors or design, however it adjusts the margins and the paddings of your main body content so your main content should go inside of that. 

    <header class="swagoo-header">
        <div class="swagoo-header-title">Swagoo Framework</div>
        <div class="swagoo-header-subtitle">Mobile-First Responsive Design Framework With Pure CSS & Funky Swag</div>
    </header>

    <div class="swagoo-container">
    </div>

    <hr>

    <footer class="swagoo-footer">
        <p>Copyright © Swagoo Responsive Design Framework 2019 All Rights Reserved.</p>
    </footer>
