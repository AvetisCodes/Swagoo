Swagoo Framework
Mobile-First Responsive Design Framework With Pure CSS & Funky Swag

Welcome

Swagoo is a responsive design framework with very minial footprint and meant for prototyping. It is easily adjustable so you can start playing around. It is not meant for production quality websites as it doesn't cover the full range of elements and is not fully compatible with all browsers - only the most common ones.

It is a 6 column framework that has many column combinations to fit your needs. You can learn more about that in the Grid page. You can also learn about which elements are covered by the framework and how to use them by following the navigation panel.

It is extremely easy to install so just follow directions below and you should be ready to roll in no time.

Fun Fact

All of the documentation pages have used Swagoo so you can test the responsivness of the framework in real-time by messing around with the existing designs and layouts.
